🧪 Default ADK Dev UI

The default ADK dev UI (with full debug tools, event replay, state inspector) runs as a separate local server:

cd stargazer && adk web stargazer_agent

# Opens at http://localhost:8000

Note: The ADK dev UI is an Angular app that cannot be embedded in our custom UI. Our Agent Insights panel replicates the same data via SSE streaming.
